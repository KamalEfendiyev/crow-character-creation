# CROW — Character Builder

Интерактивный прототип системы создания персонажа для сюжетного шутера.

Проект реализован как веб-приложение на React и используется для тестирования нарративной и визуальной составляющей перед переносом в Unreal Engine.

---

## 🧠 Концепция

Игрок не просто выбирает “класс”, а формирует личность персонажа через четыре слоя:

* **Спортивная карьера** — базовая подготовка
* **Военная специализация** — поведение в бою
* **Черты характера** — психическое состояние
* **Психологические травмы** — последствия опыта

Каждый выбор сопровождается текстом из разных источников:

* записи тренера
* заметки командира
* заключение психотерапевта
* личные показания персонажа

Это создаёт ощущение досье, а не игрового меню.

---

## 🎮 Функциональность

* Пошаговый выбор (один вариант на этап)
* Интерактивные карточки с изображениями
* Описание при наведении
* Финальный экран с собранным досье
* Стилизованный интерфейс (военный/психологический тон)
