# CROW — Character Builder

Интерактивный прототип системы создания персонажа для сюжетного шутера.

Проект реализован как веб-приложение на React и используется для тестирования нарративной, визуальной и поведенческой составляющей перед переносом в Unreal Engine.

---

## 🧠 Концепция

Игрок не просто выбирает “класс”, а формирует личность персонажа через несколько слоёв:

* **Выбор персонажа** — базовая поведенческая модель (Адам / Виктор)
* **Спортивная карьера** — фундамент подготовки
* **Военная специализация** — стиль ведения боя
* **Черты характера** — психическое состояние
* **Психологические травмы** — последствия опыта

Каждый выбор влияет не только на описание, но и на визуальное представление (изображения, эффекты, поведение интерфейса).

---

## 🎮 Функциональность

* Пошаговый выбор (один вариант на этап)
* Выбор персонажа с уникальной подачей (Адам / Виктор)
* Динамическая подмена изображений в зависимости от персонажа
* Интерактивные карточки с визуальными эффектами:
  * Адам — “сбой”, нестабильность
  * Виктор — давление, тяжесть, доминирование
* Описание при наведении с эффектом печати (typewriter)
* Кнопки навигации (вперёд / назад / в меню)
* Финальный экран с собранным досье
* Уникальные визуальные рамки в зависимости от персонажа

---

## 🧾 Нарративная подача

Каждый этап оформлен как часть досье:

* характеристика классного руководителя
* записи тренера
* заметки командира
* заключение психотерапевта
* личные показания

Тексты написаны в разных стилях, что создаёт ощущение реального документа, а не игрового меню.

---

## 🎨 UI и атмосфера

Интерфейс стилизован под:

* военные отчёты
* психологические заключения
* архивные досье

Особенности:

* моноширинные шрифты
* приглушённая цветовая палитра
* эффекты “нестабильности” и “давления”
* минимализм без явных игровых элементов
